# MuleTube

A creative React.js YouTube clone for the muLearn Web Development task.

MuleTube keeps the familiar video-platform structure, but avoids the default tutorial-clone look. The interface is styled like an indie media studio console with sharp typography, dark editorial surfaces, cinematic thumbnails, and compact creator controls.

## Features

- Responsive YouTube-style app shell with sidebar, search, categories, feed, and watch queue
- Search and topic filtering for videos
- Clickable video cards that update the featured player and queue
- Material UI components and icons, customized with a distinct CSS system
- No remote media dependency: thumbnails are code-generated visual panels
- Mobile layout with collapsible density and touch-friendly controls

## Tech Stack

- React.js
- Vite
- Material UI
- CSS custom properties

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL printed by Vite.

## Build

```bash
npm run build
```

## Hosting

- Vercel: use the default `npm run build`
- GitHub Pages: use `npm run build:pages`

## Project Note

The task asked for a YouTube clone using React.js and Material UI. This project keeps that core requirement while pushing the visual direction beyond a generic red-and-white clone.
